GridapPETSc.PetscInitializeNoPointers! — Method
function PetscInitializeNoPointers!(args::Vector{String}, filename::String, help::String)

Calls PetscInitialize() without the pointers to argc and args. This is called only by the PETSc Julia interface. Even though it might start MPI it sets the flag to indicate that it did NOT start MPI so that the PetscFinalize() does not end MPI, thus allowing PetscInitialize() to be called multiple times from Julia without the problem of trying to initialize MPI more than once.
